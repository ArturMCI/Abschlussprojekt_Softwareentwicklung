# Abschlussprojekt_Softwareentwicklung

## Konkreter 4-Meilenstein-Plan
### Milestone 1 (Tag 1–2): Solver-Minimum

- harte codierte Mini-Struktur (z.B. 4 Knoten, paar Federn)
- K aufbauen, Randbedingungen, solve
- Plot original + deformiert

### Milestone 2: Generator + UI

- Rechteck-Grid erzeugen (nx×ny Knoten)
- Federn zu Nachbarn (horizontal/vertikal + optional diagonal)
- Streamlit: Parameter einstellen, Solve drücken, Plot sehen

### Milestone 3: Optimierung

- Federenergie, Knotenscore
- iteratives Entfernen bis Zielmasse
- Connectivity check

### Milestone 4: Save/Load + MBB-Demo + Doku

- JSON Export/Import
- MBB-Fall als Preset-Button
- README, requirements, Screenshots/GIF
